    Route Distance Capture — Help

    Purpose

    This tool builds a rally route, calculates routed road distance, adds GPS control points, draws red trip lines, checks locations externally, and exports CSV data.

    Blue
    routed road path
    Red
    control trip line
    Pin
    waypoint or control point
    Green
    fit route view

    Fit Route

    Use Fit Route to zoom the map so the whole current route is visible.

    Add Next WP

    Use Add Next WP when adding the next single waypoint at the end of the route.

    1. Click Add Next WP: OFF.
    2. The button changes to ON.
    3. Click the map once.
    4. One new waypoint is added at the end of the route.
    5. The button automatically returns to OFF.
    Add Next WP = one click, one waypoint, then OFF.

    Insert After Selected

    Use Insert After Selected when adding one or more missed or intermediate waypoints.

    1. Select the waypoint after which new points should be inserted.
    2. Click Insert After Selected: OFF.
    3. The button changes to ON.
    4. Click the map to insert a waypoint after the selected waypoint.
    5. The new waypoint becomes the selected waypoint.
    6. Continue clicking the map to add more inserted waypoints in sequence.
    7. Click the button again to turn Insert mode OFF.
    Insert After Selected = stays ON and allows a series of inserted waypoints.

    Add GPS Control

    Use Add GPS Control when adding a control point from known GPS coordinates.

    1. Enter the control GPS coordinate.
    2. Select the control type.
    3. Choose whether to add as next WP or insert after an existing WP.
    4. Optionally enter line start and end GPS coordinates.
    5. Add a note if required.
    6. Click Add WP.

    Control types include Instruction, STC, Start STC, Marshal, and Open Control.

    Trip Lines

    A trip line is shown in red. It represents the timing line across the road.

    The visible red thickness is only for display. The real trip line is the centre line between the two entered coordinates.

    If trip line coordinates are entered, both Line start GPS and Line end GPS must be valid.

    Waypoint cards with a trip line show LINE distance to 3 decimal places, for metre-level display.

    Edit Control / Trip Line

    1. Click the waypoint pin on the map.
    2. Click Edit Control / Trip Line in the popup.
    3. Update the GPS, type, trip line, or note.
    4. Click Update WP.

    The waypoint keeps the same number. If the waypoint GPS changes, the route recalculates.

    Street View, Satellite View, and Google Earth

    The popup buttons open the selected waypoint directly in external map tools. Street View is useful for checking the exact control area without manually searching in Google Maps.

    Delete Waypoint

    Use Delete Waypoint from the map popup. Deleting is deliberate and should be used carefully.

    Distance Display

    • SEG and CUM are shown to 2 decimals.
    • LINE is shown to 3 decimals when a trip line exists.

    Save Route / Load Route

    Save Route saves the full editable route as a JSON project file. Choose where to save it, such as a rally folder for Day 1 or Day 2.

    Load Route opens a previously saved JSON route file and redraws the waypoints, route, and red trip lines.

    JSON is for editable route projects. CSV is for exporting route/control data to other rally workflows.

    CSV Export

    Export CSV saves waypoint number, type, segment distance, cumulative distance, line distance, GPS coordinates, trip line coordinates, and notes.

    Reset

    Reset clears the current route, waypoints, control points, trip lines, and notes.

    Suggested Working Method

    1. Add the main route waypoints in order.
    2. Use Insert After Selected to improve the route shape.
    3. Add GPS controls.
    4. Add trip line start and end coordinates.
    5. Check each control using Street View or Satellite View.
    6. Confirm distances.
    7. Export CSV.

    Important Note

    The tool uses routed road distance, not straight-line distance. The route depends on available map and routing data.

    Future versions may calculate the exact distance where the blue route crosses the red trip line.
